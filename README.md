<h1> Parallel bubble sort </h1>
<p> Parallel bubble sort versions on: </p>
<ul>
  <li> CUDA </li>
  <li> MPI </li>
  <li> OpenMP </li>
</ul>


<h2> Compilation requirement </h2>
<ul>
  <li> CUDA Toolkit 10 or superior</li>
  <li> Qt 5.8 </li>
</ul>

<p> In <em> src </em> folder, there is a Qt Creator project. </p>
